# Car Crash
An interactive car game where the player has to avoid hitting the opponent's cars. Will you get the highest score?

## Languages and Frameworks

### Front-End
CSS
HTML
JavaScript

## Installation
First, clone/fork-clone this repo. Once in the directory, in the terminal run:

```
open index.html
```
